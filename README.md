#infoex

第一次现代数据库作业：信息抽取

#使用开源框架情况

http框架： apache httpclient<br>
html解析：jsoup<br>
xml框架： dom4j<br>
NLP框架： fnlp (复旦大学自然语言处理) 


#构建说明

下载maven工程源代码，在工程跟目录（infoex/）下执行maven命令：  mvn install<br>
因为使用的NLP库的模型文件较大没法直接上传到网络学堂，所以需要从网络云盘中下载 <a>http://pan.baidu.com/s/1mhKNWvQ</a><br>
将下载好的models文件夹放入工程跟目录中。
